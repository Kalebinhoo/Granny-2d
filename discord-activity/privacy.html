Política de Privacidade - Granny 2D

Última atualização: 24 de maio de 2026

Resumo: Granny 2D não coleta, armazena ou compartilha dados pessoais dos usuários. Todos os dados de jogo são temporários e existem apenas durante a sessão ativa.

1. Informações que Coletamos

1.1 Dados Fornecidos pelo Discord

Quando você usa a Atividade através do Discord, recebemos temporariamente:

  • ID de usuário do Discord (para identificar jogadores na sessão)
  • Nome de usuário do Discord (para exibição durante o jogo)
  • Avatar do Discord (se aplicável)

Importante: Esses dados são fornecidos pelo Discord SDK e são usados apenas durante a sessão de jogo ativa. Não armazenamos essas informações.

1.2 Dados de Jogo

Durante o jogo, processamos temporariamente:

  • Posição do jogador no jogo
  • Inventário do jogador
  • Estado do jogo (sala atual, dia, etc.)

Importante: Todos esses dados existem apenas na memória do navegador durante a sessão e são descartados quando você fecha o jogo.

2. Como Usamos as Informações

Usamos as informações temporárias apenas para:

  • Permitir a funcionalidade do jogo multiplayer
  • Identificar jogadores durante a sessão
  • Sincronizar o estado do jogo entre jogadores
  • Exibir nomes de usuário durante o jogo

3. Armazenamento de Dados

NÃO armazenamos dados pessoais. Todos os dados são:

  • Processados apenas na memória do navegador
  • Temporários e existem apenas durante a sessão ativa
  • Automaticamente descartados quando você fecha o jogo
  • Não salvos em servidores ou bancos de dados

4. Compartilhamento de Dados

NÃO compartilhamos seus dados com terceiros.

Os únicos dados compartilhados são:

  • Dados de jogo temporários entre os jogadores na mesma sessão (necessário para o multiplayer)
  • Comunicação com o Discord através do Discord SDK (conforme a Política de Privacidade do Discord)

5. Cookies e Tecnologias Similares

A Atividade não usa cookies ou tecnologias de rastreamento. Podemos usar armazenamento local do navegador apenas para:

  • Salvar preferências de jogo (opcional)
  • Manter estado temporário da sessão

Esses dados permanecem no seu dispositivo e podem ser limpos a qualquer momento através das configurações do navegador.

6. Segurança

Implementamos medidas de segurança para proteger a integridade do jogo:

  • Comunicação através de HTTPS
  • Uso do Discord SDK oficial
  • Processamento de dados apenas no lado do cliente

7. Privacidade de Crianças

A Atividade está em conformidade com os Termos de Serviço do Discord, que requerem que os usuários tenham pelo menos 13 anos de idade. Não coletamos intencionalmente informações de crianças menores de 13 anos.

8. Seus Direitos

Como não armazenamos dados pessoais, não há dados para acessar, corrigir ou excluir. Todos os dados são temporários e automaticamente descartados ao final da sessão.

9. Serviços de Terceiros

A Atividade usa os seguintes serviços de terceiros:

  • Discord: Para autenticação e funcionalidade de atividade (veja a Política de Privacidade do Discord)
  • Vercel: Para hospedagem da aplicação (veja a Política de Privacidade da Vercel)

10. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos sobre mudanças significativas atualizando a data no topo desta página.

11. Contato

Se você tiver dúvidas sobre esta Política de Privacidade, entre em contato através do repositório GitHub:

https://github.com/Kalebinhoo/Granny-2d

12. Conformidade com LGPD

Esta Atividade está em conformidade com a Lei Geral de Proteção de Dados (LGPD) do Brasil, pois:

  • Não coleta dados pessoais sensíveis
  • Não armazena dados pessoais
  • Processa dados apenas para a finalidade específica do jogo
  • Não compartilha dados com terceiros (exceto conforme necessário para o funcionamento do Discord)
